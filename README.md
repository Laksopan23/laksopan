# Laksopan Ravindran | Software Engineer

A professional personal portfolio showcasing expertise in Full-Stack Development and Artificial Intelligence. This project features a high-fidelity user interface, a dynamic career timeline, and a curated project showcase.

## 🚀 Key Features

- **Modern UI/UX**: Built with React 19 and Tailwind CSS for a sleek, minimalist aesthetic.
- **Project Showcase**: Detailed documentation of key engineering projects.
- **Dynamic Timeline**: A visual representation of professional milestones and experience.
- **Responsive Design**: Optimized for all devices and screen sizes.

## 🛠️ Technology Stack

- **Frontend**: React 19, TypeScript, Tailwind CSS
- **Build Tool**: Vite 7
- **Animations**: Framer Motion
- **Icons**: Lucide React

## 📦 Getting Started

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
```

### Production Build
```bash
npm run build
```
